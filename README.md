# 💫 About Me
BTech Computer Science @ KIIT. I work on algorithms, distributed systems, and projects driven mostly by curiosity. Most of my work is exploratory — started to understand something, not to ship something.

---

## 🔬 Projects

**HBO — Highest Bidder Optimization** · *exploratory*
Original metaheuristic algorithm benchmarked against genetic algorithms on NP-hard problems. Auction-inspired search dynamics.

**Small-World RAFT** · *exploratory*
RAFT consensus variant using small-world network overlays. Explored expander graphs, Kleinberg models, fault-tolerance tradeoffs.

**Androp** · *in progress*
P2P Android document sharing with blockchain-based authenticity verification.
Stack: Java, Google Nearby Connections API, Web3j, Solidity, Ganache.

**RL Chess Engine** · *exploratory*
Reinforcement learning chess agent with a fallacy-aware Q-learning approach. React UI.

**PyPassword** · *complete*
Python password generator with HIBP breach checking.
